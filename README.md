# ⚡ IoT-Based Smart Energy Meter with Energy Optimization

## 📌 Overview
This project presents an IoT-based smart energy meter system for real-time monitoring and optimization of electricity consumption. It eliminates manual meter reading and provides intelligent alerts for efficient energy usage.

## 🚀 Features
- 🔍 Real-time energy monitoring
- 📊 IoT cloud data visualization
- 🔔 Threshold alert system (buzzer)
- ⚡ Penalty-based consumption tracking
- 📱 Remote load control
- 📉 Energy optimization

## 🛠️ Technologies Used
- ESP32 / ESP8266
- Embedded C / Arduino IDE
- IoT Cloud (ThingSpeak / Blynk)
- Sensors (Voltage & Current)

## ⚙️ Working
The system continuously monitors electricity usage and sends data to the cloud.
- If usage exceeds limit → buzzer alert 🔔
- Extra usage → penalty ⚡
- User can control load remotely 📱

## 🎯 Objectives
- Reduce energy wastage
- Improve billing accuracy
- Enable real-time monitoring
- Promote smart energy usage
